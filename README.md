oss-contributions
=================

List OSS contributions (GitHub repositories) of specified users.

```
Usage: oss_contributions [options]  [<user>...]
    -u, --user=USER
    -o, --organization=ORGANIZATION
    -m, --min-stargazers=NUM
    -c, --contribution-only
    -s, --sort=ORDER
    -r, --render=TEMPLATE
```
